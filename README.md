# film-companion
Capture film photo settings on a phone to aid improving photography

## Using the App

### First Time Setup (Need WiFi)
1. Open the URL on your phone: https://fkpza.github.io/film-companion/
2. **iOS**: Tap Share button → "Add to Home Screen"
3. **Android**: Tap Menu (⋮) → "Add to Home Screen"
4. Name it "Film Companion"
5. The app downloads and caches everything

### Offline Use (No WiFi Needed)
- Tap the home screen icon
- Works 100% offline from now on
- All data saves locally on your device
- Camera, GPS, everything works

### Light/Dark Theme
- Tap the ☀️/🌙 button (top right)
- **Light theme** = better outdoor visibility in sunlight
- **Dark theme** = better for night/indoor use
- Your choice is remembered

### Workflow
1. **New Roll** → Enter title, date, film type, ISO
2. **Add Frame** → Settings pre-populate from last frame
3. **Take Photo** (optional) → Viewfinder simulates focal length
4. **Save** → See "SAVED ✓" confirmation
5. **Export** → Download HTML + text log + JPEGs

## Basic Troubleshooting

### "Camera not working"
- Make sure you're accessing via `https://` URL (not `file://`)
- Grant camera permission when prompted
- Try refreshing the page

### "Not saving"
- Check browser storage isn't full
- Look for "SAVED ✓" notification after each save
- Try clearing some old data if needed

### "Can't add to home screen"
- iOS: Must use Safari (not Chrome)
- Android: Use Chrome or Firefox
- Make sure you're on the GitHub Pages URL

### "Need to update the app"
- Delete from home screen
- Clear browser cache
- Re-visit URL and re-add to home screen
